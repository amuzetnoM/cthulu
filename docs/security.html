Security & Configuration

Environment Overrides
Environment variables override specific fields in `config.json`. Avoid storing credentials inside config files — use per-field overrides (e.g., `MT5_PASSWORD`).
Secrets Scanning
Pre-commit includes `detect-secrets`. Use `git-secrets` in CI and rotate credentials regularly.
Operational Notes
Bind RPC to localhost by default, avoid exposing MT5 terminal on public interfaces, and use OS-level service managers (systemd/nssm) for auto-restarts with restricted permissions.
