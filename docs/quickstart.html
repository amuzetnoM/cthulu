Quick Start

Three steps to run in dry-run mode and inspect results quickly.

1. Clone & Setup 
git clone https://github.com/amuzetnoM/herald.git
cd herald
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
2. Configure

Use `config.example.json` then override sensitive values with environment variables.

copy config.example.json config.json
set MT5_LOGIN=12345
set MT5_PASSWORD=secret
3. Dry Run

Run without placing real orders to validate signals and indicators.

python __main__.py --config config.json --dry-run
