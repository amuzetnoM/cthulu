Herald v2.0.0 — 2024-12-06

Highlights
- Autonomous Trading Loop: Complete implementation of autonomous trade execution.
- Technical Indicators: Full suite of professional-grade indicators (RSI, MACD, Bollinger, Stochastic, ADX).
- Position Management: Sophisticated position sizing and risk management.
- Exit Strategies: Multi-layered exit logic with stop-loss and take-profit.

Key Changes
### Added
- Technical Indicators Library (`indicators/`): RSI, MACD, Bollinger Bands, Stochastic Oscillator, ADX.
- Position Management System (`position/`): automated sizing, real-time tracking, multi-timeframe analysis.
- Execution Engine (`execution/`): order placement, slippage control, market/pending order support.
- Exit Strategies (`exit/`): fixed stop-loss/take-profit, trailing stop, time-based exits, risk-based sizing.

Enhanced
- MT5 integration improvements, expanded configuration options, and enhanced logging.

Fixed
- MT5 connection and synchronization issues, memory leakage in continuous operation.

Migration Notes
- Config format updated — review `config.example.json`.
- DB: Automatic schema migration from v1.0.0.
- Updated `requirements.txt`.

Known Issues
- Edge cases in high-frequency conditions
- Limited backtesting capabilities (planned for v3.0.0)

What's Next
- v3.0.0 to focus on production readiness and comprehensive testing
